# xDownload Manager

Trình quản lý tải xuống cho macOS — tải nhanh, đa kết nối, hỗ trợ HLS/video, kèm extension cho Chrome.

## Tải xuống

Vào mục **[Releases](../../releases/latest)** và tải file `.dmg` mới nhất.

## Cài đặt

1. Mở file `.dmg`, kéo **xDownload Manager** vào thư mục **Applications**.
2. Lần đầu mở, nếu macOS báo *"bị hỏng"* / *"không xác minh được nhà phát triển"* (app chưa notarize), chạy lệnh sau trong Terminal rồi mở lại:

   ```bash
   xattr -cr "/Applications/xDownload Manager.app"
   ```

   Hoặc: chuột phải vào app → **Mở**.

## Yêu cầu

- macOS (Apple Silicon / Intel)
